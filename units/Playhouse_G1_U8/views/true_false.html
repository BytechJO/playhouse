Adjectives are used to describe nouns. Often, more than one adjective is used to describe a noun. When this occurs, there is a proper order for the adjectives. The order is given above. When only two adjectives are used together, separate them with a comma or “and.” Use commas to separate three or more adjectives in a series. When there are only two adjectives before a noun, the adjectives that describe color, size, or age do not need to be separated by commas. Do not use a comma or “and” to separate a common adjective from a proper adjective: a small African elephant
